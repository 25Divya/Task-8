# Task-8
Simple Sales Dashboard Design

## Objective:
Create a Power BI dashboard showing sales performance by product category, region, and month.

## Tools:
- Power BI Desktop
- Dataset:eCommerce sales data

## Visuals:
- Line Chart: Monthly Sales Trend
- Bar Chart: Sales by Region
- Donut Chart: Sales by Category
- Slicer: Region filter

## Key Insights:
1. West region had the highest sales in Q3.
2. Technology category outperformed others in revenue.
3. Peak sales occurred in December 2022.
4. Central region needs improvement.

## Files Included:
- `Ecommerce_project.pbix`: Power BI file
- `insights.txt`: Key findings
